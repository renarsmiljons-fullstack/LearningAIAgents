# Deep Agents

Exploration workspace for building AI agents with [LangChain](https://js.langchain.com/), [LangGraph](https://langchain-ai.github.io/langgraphjs/), and the [DeepAgents](https://www.npmjs.com/package/deepagents) framework.

## Getting Started

Requires Node.js >= 20.

```bash
npm run setup
```

This installs all dependencies and creates a `.env` file from `.env.example`. Add your API key to `.env` before running. See `package.json` for available scripts.
