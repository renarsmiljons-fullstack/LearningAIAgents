# LearningAIAgents

A sandbox environment for exploring, experimenting with, and learning about AI agent frameworks and services.

## Purpose

This repository is a personal learning space — a place to build small projects, run experiments, and develop a deeper understanding of how AI agents work across different frameworks and paradigms.

## Frameworks & Tools of Interest

- **LangChain** — foundational framework for building LLM-powered applications
- **LangGraph** — graph-based orchestration for multi-step agent workflows
- **Vercel AI SDK** — streaming-first toolkit for building AI-powered UIs
- **DeepAgents** — emerging framework for deep agent architectures
- **Trigger.dev** — background job infrastructure that can power agent pipelines
- ...and whatever else comes next

The list will grow as the ecosystem evolves. The goal isn't to master every framework but to understand the trade-offs, patterns, and mental models behind building AI agents.

## Structure

Each exploration or experiment lives in its own directory. There's no enforced structure — this is a sandbox, not a monorepo.
